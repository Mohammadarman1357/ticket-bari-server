## 🎫 TicketBari | Online Ticket Booking Platform
TicketBari is a comprehensive MERN stack application designed to streamline the process of discovering and booking travel tickets (Bus, Train, Launch, Plane, etc.). The platform features a multi-role system (User, Vendor, Admin) with secure payment integration and real-time ticket management.

## 📖 Project Overview
The Online Ticket Booking Platform provides a seamless experience for travelers to book their journeys. While Users can search, filter, and pay for tickets, Vendors can manage their own inventory, and Admins oversee the entire ecosystem, ensuring a secure and fraud-free environment.
________________________________________
## ✨ Key Features
## 👤 User Features

*	Smart Searching: Search tickets by "From" and "To" locations.
*	Advanced Filtering: Filter tickets by transport type and sort by price (Low to High / High to Low).
*	Secure Booking: Book tickets with a live countdown timer for departure.
*	Stripe Integration: Secure and seamless payment gateway.
*	Dashboard: Manage bookings, view transaction history.
*	Dark/Light Mode: Personalized UI experience with theme toggling.

## 🏢 Vendor Features

*	Ticket Management: Add, update, and delete tickets (with Image hosting via ImgBB).
*	Booking Requests: Accept or reject user booking requests.
*	Revenue Analytics: Interactive charts for Total Revenue, Tickets Sold, and Inventory status.

## 🛠 Admin Features
*	Ticket Moderation: Approve or reject vendor-added tickets.
*	User Control: Manage roles (Admin/Vendor) and mark fraudulent vendors.
*	Advertisement System: Toggle featured tickets for the homepage (Max 6).

## 🛡️ Security Measures

* JWT Authentication: Protected API routes to ensure only authorized users can access sensitive data.
* Environment Variables: All Firebase and MongoDB credentials are hidden using .env.
* Private Routes: Logged-in users are persisted on reload, and unauthorized access is restricted.
